# 试纸色差检测系统 使用手册

## 环境准备

### 创建虚拟环境

```bash
# 在项目根目录下创建虚拟环境
python3 -m venv venv

# 激活虚拟环境
source venv/bin/activate    # macOS / Linux
# venv\Scripts\activate     # Windows
```

### 依赖安装

```bash
# 确保已激活虚拟环境后执行
pip install -r backend/requirements.txt
```

### 海康工业相机 SDK 安装（可选）

如需使用海康工业相机（MV-CU060-10GC 等 GigE Vision 型号），需额外安装 MVS SDK：

**Windows：**
1. 从[海康机器人官网](https://www.hikrobotics.com/cn/machinevision/service/download)下载 MVS 安装包并安装
2. 将 SDK Python 封装文件复制到项目中：
```bash
# 源目录
C:\Program Files (x86)\MVS\Development\Samples\Python\MvImport\

# 复制到
backend/camera/MvImport/
```

**Linux：**
1. 下载并安装 MVS（`.deb` 或 `.tar.gz`）
2. 复制 Python 封装文件：
```bash
cp -r /opt/MVS/Samples/64/Python/MvImport/ backend/camera/MvImport/
```
3. 设置动态库路径：
```bash
export LD_LIBRARY_PATH=/opt/MVS/lib/64:$LD_LIBRARY_PATH
```

> **注意**：未安装 MVS SDK 不影响单图分析和模拟相机测试功能，仅海康相机模式不可用。

### 启动服务

**终端 1 — 后端：**
```bash
source venv/bin/activate
cd backend
python3 app.py
```
启动后显示 `Running on http://0.0.0.0:5002`

**终端 2 — 前端：**
```bash
cd frontend
python3 -m http.server 8080
```

打开浏览器访问：`http://localhost:8080`

## 功能一：单图分析

### 界面说明

```
┌─────────────────────────────────────────────────────┐
│                  试纸色差检测系统                     │
├──────────────┬──────────────────────────────────────┤
│  ① 参数配置  │  ③ 步骤条（点击可切换步骤）           │
│  ② 上传图片  │  ④ 步骤内容（图片 + 数据）            │
│  ⑤ 开始分析  │  ⑥ 结果卡片（底部）                  │
└──────────────┴──────────────────────────────────────┘
```

### 操作步骤

#### 第一步：设置参数

**ΔE 阈值**（默认 5）
- 拖动滑块调整，范围 0~30
- 含义：网格色差超过此值则判定为变色
- 建议：根据试纸灵敏度调整，一般 3~10 之间

**网格数量**（默认 50）
- 输入框直接填写，范围 3~200
- 含义：将试纸划分为多少个网格进行统计
- 建议：50 是精度与速度的平衡点，精度要求高可设 100

#### 第二步：上传图片

- 点击上传区域，或将图片**拖拽**到上传区
- 支持格式：JPG、PNG
- 上传成功后显示图片预览
- 如需更换图片，点击预览图即可重新上传

> **拍摄建议**：
> - 试纸平放，避免弯曲
> - 光线均匀，避免强烈阴影
> - 试纸占画面 60% 以上
> - 确保定位孔（黑洞）清晰可见

#### 第三步：开始分析

点击「开始分析」按钮，等待进度条完成（通常 2~5 秒）。

### 查看结果

#### 步骤条

分析完成后，顶部出现 4 个步骤节点：

| 步骤 | 内容 |
|------|------|
| Step 1 ROI 提取 | 查看试纸轮廓检测和校正结果 |
| Step 2 黑洞裁剪 | 查看定位孔识别和裁剪效果 |
| Step 3 分割与色差 | 查看左右分割线、LAB 均值、整体 ΔE |
| Step 4 网格统计 | 查看热力图和变色标注图 |

- 点击任意步骤节点可跳转查看
- 也可点击「上一步 / 下一步」按钮顺序浏览

#### 结果卡片（底部）

| 卡片 | 说明 |
|------|------|
| 左侧平均 LAB | 对照区的 L、a、b 值，作为基准参考色 |
| 右侧平均 LAB | 反应区的 L、a、b 值 |
| 整体 ΔE | 左右均值的色差，反映整体反应程度 |
| 超阈值比例 | 变色网格数 ÷ 总网格数，反映变色面积占比 |

## 功能二：视频流检测

视频流检测支持三种视频源模式，适用于不同场景。

### 视频源模式

#### 模式 1：视频文件 / RTSP

适用于本地视频文件回放或 RTSP 流检测。

1. 视频源类型选择 **「视频文件 / RTSP」**
2. 在地址栏输入视频文件路径（如 `/path/to/video.mp4`）或 RTSP 地址（如 `rtsp://192.168.1.100:554/stream`）
3. 设置抓拍间隔和检测参数
4. 点击「开始检测」

#### 模式 2：海康工业相机

适用于接入海康 MV-CU060-10GC 等 GigE Vision 工业相机的生产环境。

**前提条件**：已安装 MVS SDK 并复制 MvImport 文件（见「环境准备」）。

1. 视频源类型选择 **「海康工业相机」**
2. 点击 **「刷新」** 按钮扫描网络中的相机设备
3. 在下拉列表中选择目标相机
4. 设置曝光时间（默认 10000μs，根据光线调整）
5. 设置抓拍间隔和检测参数
6. 点击「开始检测」

> **相机连接提示**：
> - 确保相机与工控机在同一网段
> - 首次使用建议先通过 MVS 客户端验证相机连通性
> - 如果刷新后未发现设备，检查网线连接和 IP 配置

#### 模式 3：模拟相机（测试）

适用于没有实体相机时的开发测试，从本地测试图片循环读取模拟相机抓图。

1. 视频源类型选择 **「模拟相机（测试）」**
2. 设置 **图片切换间隔**（默认 1 秒，控制多久切换到下一张测试图片）
3. 设置抓拍间隔和检测参数
4. 点击「开始检测」

> 测试图片位于 `mediamtx/test-img/` 目录，系统会循环读取其中的图片。

### 检测参数

| 参数 | 默认值 | 说明 |
|------|--------|------|
| 抓拍间隔 | 5 秒 | 每隔多少秒进行一次色差分析 |
| ΔE 阈值 | 5 | 色差超过此值判定为变色 |
| 网格数 | 50 | 试纸划分的网格数量 |
| ROI 评分权重 | 矩形度5/顶点5/长宽比20/面积0/亮度95 | 点击展开可调 |

### 检测过程

- 右侧显示实时预览画面
- 下方时间轴逐帧显示帧卡片（缩略图 + ΔE 值 + 变色比例）
- 点击帧卡片的 **「详情」** 按钮可查看该帧的完整分步分析
- 当某帧 ΔE 超过阈值时自动判定为变色，停止检测并显示最终结果
- 也可随时点击 **「停止检测」** 手动终止

## 结果解读

### ΔE 参考范围

| ΔE 值 | 人眼感知 | 含义 |
|-------|---------|------|
| 0~1 | 几乎无差异 | 对照区与反应区颜色基本一致，未反应 |
| 1~3 | 轻微差异 | 轻微变色，反应较弱 |
| 3~6 | 明显差异 | 明显变色，反应中等 |
| > 6 | 显著差异 | 显著变色，反应强烈 |

### 变色比例解读

- **比例 = 超阈值网格数 ÷ 全图总网格数**
- 分子：右侧（反应区）中 ΔE 超过阈值的网格数
- 分母：左右两侧全图总网格数
- 例：比例 24% 表示全图约 1/4 的区域发生了超阈值变色

### 热力图颜色说明

- 蓝色：ΔE 低，颜色接近对照区
- 红色：ΔE 高，颜色偏离对照区明显
- 变色标注图中红色高亮区域为超过阈值的网格

## 常见问题

**Q：分析失败，提示"未检测到试纸区域"**
- 检查图片是否清晰，试纸是否占画面主体
- 尝试在光线更均匀的环境下重新拍摄

**Q：黑洞没有被正确识别，裁剪位置不对**
- 确保定位孔（黑洞）在图片中清晰可见，颜色足够深
- 检查图片是否过曝或欠曝

**Q：变色比例异常偏高或偏低**
- 调整 ΔE 阈值：阈值越小，判定为变色的网格越多；阈值越大，越少
- 增大网格数量可提高统计精度

**Q：后端启动报错 `Address already in use`**
- 5002 端口被占用，执行以下命令杀掉占用进程：
```bash
kill -9 $(lsof -ti:5002)
```

**Q：选择海康相机模式后提示"MVS SDK 未安装"**
- 请按照「环境准备 > 海康工业相机 SDK 安装」步骤安装 MVS 并复制 MvImport 文件
- 未安装 SDK 不影响其他功能的使用

**Q：刷新设备列表后显示"未发现相机设备"**
- 检查相机电源是否开启、网线是否连接
- 确认相机与工控机在同一网段
- 尝试通过 MVS 客户端确认相机是否可被发现
