# Windows客户交付方案

## 方案一：PyInstaller打包（推荐）

### 优点
- 单个exe文件，无需Python环境
- 客户双击即可运行
- 包含所有依赖

### 打包步骤

#### 1. 安装PyInstaller

```bash
pip install pyinstaller
```

#### 2. 创建打包脚本

在项目根目录创建 `build.spec` 文件（见下方）

#### 3. 执行打包

```bash
pyinstaller build.spec
```

#### 4. 交付内容

打包完成后，将以下内容打包成zip交给客户：

```
shijuedemo_release/
├── shijuedemo.exe          # 主程序
├── frontend/               # 前端文件
│   ├── index.html
│   ├── css/
│   └── js/
└── 使用说明.txt
```

### 使用方法

客户只需：
1. 解压zip文件
2. 双击 `shijuedemo.exe`
3. 浏览器自动打开 http://localhost:8080

## 方案二：Docker容器

### 优点
- 跨平台
- 环境隔离
- 易于部署

### 交付内容

提供 `docker-compose.yml` 文件，客户安装Docker Desktop后一键启动。

---

## 方案三：在线部署

### 优点
- 无需客户安装任何软件
- 通过浏览器访问
- 易于维护更新

### 部署平台
- Heroku
- Railway
- Vercel（前端） + Render（后端）

---

## 推荐方案

**方案一（PyInstaller）** 最适合Windows客户，操作简单，无需额外环境。
