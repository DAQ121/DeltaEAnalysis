# 试纸色差检测技术方案

## 项目概述

检测试纸变色区域与对照区域的色差，通过 ΔE 值和变色比例量化反应结果。

### 判断逻辑
- 左侧为**对照区**（未反应），右侧为**反应区**
- 以左侧平均 LAB 为基准，计算右侧每个网格的 ΔE
- 超过阈值的网格占全图总网格的比例即为变色比例

## 核心流程（v2.1）

```
原始图像
    ↓
Step 1: ROI 提取（多策略二值化 + 矩形度评分 + 透视变换校正）
    ↓
Step 2: 黑洞定位与裁剪（裁掉定位孔及其左侧）
    ↓
Step 3: 左右分割 + LAB 均值计算 + 整体 ΔE
    ↓
Step 4: 全图网格划分 → 右侧网格 ΔE 统计 → 变色比例
```

---

## 各步骤详解

### Step 1：ROI 提取（多策略二值化 + 矩形度评分）

**目标**：从原始图像中精确提取试纸区域并校正角度，即使在光照不均匀的条件下也能可靠识别矩形试纸。

**问题背景**：单一 Otsu 全局阈值在光照不均时，会把高亮区域误判为前景，导致选中光斑而非试纸。

**解决方案**：采用三种二值化策略并行生成候选轮廓，再按矩形相似度评分选出最佳试纸轮廓。

```
原始图片
    ↓ 灰度化 + 高斯模糊（7×7）
    ↓ 三种策略并行生成候选轮廓：
    │   ├─ 策略 A：自适应高斯阈值（blockSize=51, C=10）+ 正/反极性
    │   │           → 按局部窗口计算阈值，天然抗光照不均
    │   ├─ 策略 B：Canny 边缘检测（30, 100）+ 形态学闭合
    │   │           → 边缘对渐变光照鲁棒
    │   └─ 策略 C：Otsu 全局阈值（原方法）
    │               → 均匀光照下最优，保证向后兼容
    ↓ 对所有候选轮廓按矩形度评分（0~1 分）：
    │   ├─ 矩形度 40%：轮廓面积 / 最小外接矩形面积（矩形≈1.0）
    │   ├─ 顶点数 30%：多边形近似顶点数（4 为满分）
    │   ├─ 长宽比 15%：1.5~10.0 范围得满分
    │   └─ 面积占比 15%：排除过小噪点和过大光照区域
    ↓ 选取最高分轮廓 → cv2.minAreaRect() 最小旋转矩形
    ↓ cv2.getPerspectiveTransform() 透视变换（BORDER_REPLICATE 避免黑边）
    ↓ 掩码二值化阈值处理（消除插值伪影）
    ↓ 输出：摆正的试纸图像 + 掩码
```

---

### Step 2：黑洞定位与裁剪

**目标**：找到定位孔（黑洞），裁掉黑洞及其左侧无效区域

```
ROI 图像
    ↓ 灰度阈值（< 50）检测暗区域
    ↓ 与试纸掩码取交集
    ↓ 形态学开运算去噪（5×5 核）
    ↓ 取面积最大的黑洞轮廓
    ↓ 计算右边界 x + 膨胀 padding（孔宽 × 0.5）
    ↓ 裁剪：保留右边界以右的区域
    ↓ 输出：裁剪后图像 + 掩码
```

**膨胀 padding 的作用**：确保黑洞边缘的深灰色圆圈也被完全裁掉，不干扰对照区计算。

---

### Step 3：左右分割 + LAB 均值 + 整体 ΔE

**目标**：将裁剪后图像从中间分为对照区（左）和反应区（右），分别计算平均 LAB，并计算整体 ΔE

```
裁剪后图像
    ↓ 从 x = width // 2 切分
    ↓ 左侧：取掩码内所有像素 → 转 LAB → 求均值 [L₁, a₁, b₁]
    ↓ 右侧：取掩码内所有像素 → 转 LAB → 求均值 [L₂, a₂, b₂]
    ↓ 整体 ΔE = √((L₂-L₁)² + (a₂-a₁)² + (b₂-b₁)²)
```

**色差公式**：CIE76（欧氏距离）
```
ΔE = √((L₂-L₁)² + (a₂-a₁)² + (b₂-b₁)²)
```

---

### Step 4：网格划分 + 变色比例统计

**目标**：对完整裁剪图划网格，统计右侧超阈值网格占全图比例

```
裁剪后完整图像
    ↓ 按 grid_size 划分正方形网格
    ↓ 对每个网格：
        - 转 LAB，取掩码内像素均值
        - ΔE_i = √((L_i - L₁)² + (a_i - a₁)² + (b_i - b₁)²)
    ↓ 生成 ΔE 矩阵 + 热力图
    ↓ 仅统计右侧（x ≥ mid_x）中 ΔE > 阈值 的网格数
    ↓ 变色比例 = 超阈值网格数 / 全图总网格数
```

**分母说明**：分母为左右两侧总网格数，分子仅为右侧超阈值网格数，体现反应区相对于整体的变色程度。

---

## 关键参数

| 参数 | 默认值 | 说明 |
|------|--------|------|
| ΔE 阈值 | 5 | 判断网格是否变色的临界值，可在界面调整（0~30） |
| 网格大小 | 50 | 划分网格的数量，越大精度越高，可在界面调整（3~200） |
| 黑洞检测灰度阈值 | 50 | 低于此值的像素视为黑洞 |
| 黑洞膨胀 padding | 孔宽 × 0.5 | 裁剪时额外向右扩展的像素数 |

    "image": "data:image/png;base64,...",
    "threshold": 5,
    "grid_size": 50

## 鲁棒性设计

| 异常情况 | 处理策略 |
|---------|---------|
| 未检测到试纸轮廓 | 返回错误提示 |
| 未找到黑洞 | 跳过裁剪，使用完整 ROI |
| 网格内无有效像素 | ΔE 记为 0，不计入变色统计 |
