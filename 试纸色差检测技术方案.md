# 试纸色差检测技术方案

## 项目概述

### 核心需求
检测试纸变色区域与正常区域的色差，通过色差值（ΔE）和变色比例判断实验是否成功。

### 判断标准
- **成功**：色差 ΔE ≥ 设定阈值
- **失败**：色差 ΔE < 设定阈值

## 核心流程（优化版）

### 核心思路
**双图像策略**：使用增强图像进行区域识别，使用原始图像进行色差计算

```
原始图像 ──┬──→ 增强图像（用于区域识别）
           │         ↓
           │    K-means聚类分割
           │         ↓
           │    变色区域掩码
           │         ↓
           └──→ 原始图像（用掩码提取区域，计算真实色差）
```

### 阶段1：ROI提取（试纸区域定位）

**目标**：从原始图像中精确提取试纸区域并校正角度

**步骤**：
```
原始图片
    ↓
灰度化 + 高斯模糊去噪（7×7）
    ↓
Otsu自动阈值二值化
    ↓
形态学闭运算（3×3核）
    ↓
轮廓检测 → 筛选最大轮廓
    ↓
最小旋转矩形拟合
    ↓
透视变换校正角度
    ↓
输出：摆正的试纸图像 + 掩码
```

**关键技术**：
- 使用 `cv2.minAreaRect()` 获取最小旋转矩形
- 使用 `cv2.getPerspectiveTransform()` 进行透视变换
- 生成掩码用于后续只计算试纸内像素
    ↓
裁剪ROI区域
    ↓
输出：试纸区域图像
```

**关键参数**：
- 高斯核大小：`(5, 5)`
- 形态学核大小：`5×5`
- 最小轮廓面积：图像总面积的 10%

---

### 阶段2：图像增强与参考色提取

**目标**：增强图像对比度，智能识别正常区域并提取参考色

**步骤**：
```
原始ROI图像 ──┬──→ 保存原图（用于最终色差计算）
              │
              └──→ 图像增强
                      ↓
                  CLAHE光照校正（消除阴影和光照不均）
                      ↓
                  提取掩码内所有像素
                      ↓
                  K-means聚类（K=2）
                      ↓
                  识别正常区域（像素数量更多的类）
                      ↓
                  计算参考色LAB均值
                      ↓
                  用参考色填充黑洞
```

**关键技术**：
- **CLAHE（对比度受限自适应直方图均衡化）**：
  - 消除光照不均和阴影影响
  - 参数：`clipLimit=2.0, tileGridSize=(8,8)`

- **K-means聚类**：
  - 将试纸像素分为2类：正常区域和变色区域
  - 选择像素数量更多的类作为正常区域
  - 避免手动指定参考区域位置

- **黑洞填充**：
  - 检测试纸中的黑洞（如打孔）
  - 用参考色填充，避免干扰色差计算

**输出**：
- 参考色LAB值
- 填充黑洞后的图像
- 参考色色块（可视化）

---

### 阶段3：自适应网格划分

**目标**：将试纸划分为正方形网格

**步骤**：
```
填充后的ROI图像
    ↓
计算最短边尺寸
    ↓
cell_size = min(h, w) / grid_size
    ↓
按正方形网格划分
    ↓
绘制网格线（黄色）
```

**关键技术**：
- 自适应正方形网格：确保每个格子是正方形
- 网格数量根据图像尺寸自动调整

**输出**：
- 网格叠加图像
- 网格尺寸信息（行数、列数、格子大小）

---

### 阶段4：色差计算（使用原始图像）

**目标**：在原始图像上计算每个网格的真实色差

**步骤**：
```
原始ROI图像（未增强）
    ↓
按网格划分
    ↓
对每个网格：
  - 转换为CIELAB色彩空间
  - 只计算掩码内的像素
  - 计算均值 (L_i, a_i, b_i)
  - 计算色差：ΔE_i = √[(L_i - L_ref)² + (a_i - a_ref)² + (b_i - b_ref)²]
    ↓
生成色差矩阵
    ↓
生成热力图（蓝→红）
```

**关键技术**：
- **使用原始图像计算**：避免增强带来的色彩失真
- **掩码过滤**：只计算试纸内的像素，排除白色填充区域
- **CIE76色差公式**：符合人眼感知的色差计算

**输出**：
- ΔE矩阵
- 色差热力图
- ΔE值范围

### 阶段5：变色区域识别与结果统计

**目标**：识别变色区域并统计结果

**步骤**：
```
ΔE矩阵 + 阈值
    ↓
标记 ΔE > 阈值 的网格为变色区域
    ↓
生成红色遮罩叠加图
    ↓
统计：
  - 变色网格数
  - 变色比例 = 变色网格数 / 总网格数
  - 整体平均ΔE
  - 变色区域平均ΔE
    ↓
判断实验结果（成功/失败）
```

**输出内容**：
1. **变色比例**：变色区域占总面积的百分比
2. **整体平均 ΔE**：所有网格的平均色差
3. **变色区域平均 ΔE**：超过阈值的网格的平均色差
4. **实验结果**：成功（ΔE ≥ 阈值）或失败（ΔE < 阈值）
5. **可视化图像**：变色区域高亮标注

## 核心技术优势

### 1. 双图像策略
- **增强图像**：用于区域识别和分割，提高变色区域检测准确性
- **原始图像**：用于色差计算，保证色彩数据的真实性
- **优势**：既能精准识别变色区域，又能准确计算真实色差

### 2. 智能参考色提取（K-means聚类）
- 自动将试纸分为正常区域和变色区域
- 选择像素数量更多的类作为正常区域
- 无需手动指定参考区域位置
- 不受光照不均和阴影干扰

### 3. CLAHE光照校正
- 消除光照不均和阴影影响
- 增强变色区域对比度
- 提高区域识别准确性

### 4. 自适应正方形网格
- 根据试纸尺寸自动调整网格大小
- 确保每个格子是正方形
- 提高色差计算的空间分辨率

### 5. 掩码过滤
- 只计算试纸内的像素
- 排除白色填充区域和黑洞
- 避免无关区域干扰计算结果

## 关键参数配置

| 参数名称 | 建议值 | 说明 | 可调整范围 |
|---------|--------|------|-----------|
| 高斯核大小 | `(7, 7)` | 去噪强度 | `(5, 5)` ~ `(9, 9)` |
| 形态学核大小 | `3×3` | 闭运算核大小 | `3×3` ~ `5×5` |
| CLAHE clipLimit | `2.0` | 对比度限制 | `1.0` ~ `4.0` |
| CLAHE tileGridSize | `(8, 8)` | 网格大小 | `(4, 4)` ~ `(16, 16)` |
| K-means聚类数 | `2` | 正常/变色两类 | 固定为2 |
| 黑洞检测阈值 | `50` | 灰度阈值 | `30` ~ `70` |
| 黑洞膨胀核 | `15×15` | 确保完全覆盖 | `10×10` ~ `20×20` |
| 网格大小 | `20` | 平衡精度和计算量 | `10` ~ `50` |
| ΔE 阈值 | `0-30` | 判断变色的临界值 | 根据实际实验调整 |

## 鲁棒性设计

### 异常处理

| 异常情况 | 处理策略 |
|---------|---------|
| 未检测到试纸 | 返回错误提示，建议调整拍摄角度或光照 |
| ROI 过小 | 调整二值化参数重试，或降低最小轮廓面积阈值 |
| 参考区域异常 | 使用整体色彩的众数作为参考色 |
| 网格内像素过少 | 跳过该网格或使用插值 |

### 自适应优化

| 场景 | 优化方案 |
|------|---------|
| 背景不是纯黑色 | 使用自适应阈值（`cv2.adaptiveThreshold`）替代 Otsu |
| 试纸有旋转 | 使用旋转矩形（`cv2.minAreaRect`）校正角度 |
| 光照不均 | 先做直方图均衡化（`cv2.equalizeHist`）或 CLAHE |
| 试纸边缘模糊 | 增强边缘检测（Canny + 霍夫变换） |

## 输出格式

### JSON 格式
```json
{
    "success": true,
    "color_change_ratio": 0.35,
    "overall_delta_e": 12.5,
    "changed_area_delta_e": 18.3,
    "threshold": 10,
    "grid_size": [20, 20],
    "reference_color_lab": [85.2, 2.1, -3.5],
    "visualization_path": "heatmap.png",
    "roi_extracted": true,
    "roi_size": [800, 400]
}
```

### 字段说明

| 字段 | 类型 | 说明 |
|------|------|------|
| `success` | boolean | 是否成功提取并计算 |
| `color_change_ratio` | float | 变色比例（0-1） |
| `overall_delta_e` | float | 整体平均色差 |
| `changed_area_delta_e` | float | 变色区域平均色差 |
| `threshold` | float | 使用的 ΔE 阈值 |
| `grid_size` | array | 网格划分尺寸 [行, 列] |
| `reference_color_lab` | array | 参考色的 LAB 值 [L, a, b] |
| `visualization_path` | string | 热力图保存路径 |
| `roi_extracted` | boolean | 是否成功提取 ROI |
| `roi_size` | array | ROI 尺寸 [宽, 高] |
