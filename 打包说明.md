# Windows打包步骤

## 在Windows电脑上执行以下步骤

### 1. 安装依赖

```bash
pip install pyinstaller
```

### 2. 执行打包

双击运行 `build_windows.bat`

或在命令行执行：
```bash
pyinstaller build.spec
```

### 3. 交付给客户

打包完成后，将 `release` 文件夹打包成zip，包含：

```
release/
├── 试纸色差检测系统.exe
├── frontend/
│   ├── index.html
│   ├── css/
│   └── js/
└── 使用说明.txt
```

客户只需解压后双击exe即可使用。

## 注意事项

- 必须在Windows系统上打包
- 打包后的exe只能在Windows上运行
- 首次打包可能需要10-20分钟
- 生成的exe文件约200-300MB
