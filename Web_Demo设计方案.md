# Web Demo 设计方案

## 产品定位

面向实验人员的试纸色差检测 Web 工具，上传试纸图片后自动完成图像处理、色差计算和变色比例统计，全程可视化展示每个处理步骤。

---

## 整体布局

```
┌─────────────────────────────────────────────────────┐
│                    顶部 Header                       │
├──────────────┬──────────────────────────────────────┤
│              │  ┌──────────────────────────────────┐│
│   左侧       │  │  横向步骤条（Step 1 → 2 → 3 → 4）││
│   控制面板   │  ├──────────────────────────────────┤│
│              │  │                                  ││
│  · 参数配置  │  │       步骤内容区（图片 + 数据）   ││
│  · 图片上传  │  │                                  ││
│  · 分析按钮  │  ├──────────────────────────────────┤│
│              │  │  结果卡片（左LAB / 右LAB / ΔE / 比例）││
└──────────────┴──────────────────────────────────────┘
```

**设计原则**：
- 整页固定在视口内，不出现页面级滚动条
- 步骤内容区图片自适应缩放，不撑开容器
- 结果卡片固定在底部，始终可见

---

## 各区域说明

### 左侧控制面板（固定宽度 280px）

| 区域 | 内容 |
|------|------|
| 参数配置 | ΔE 阈值滑块（默认 5，范围 0~30）；网格数量输入框（默认 50，范围 3~200） |
| 图片上传 | 拖拽或点击上传，支持 JPG/PNG；上传后显示预览图，点击预览可重新上传 |
| 分析按钮 | 上传图片后激活，点击触发分析；分析中显示进度条动画 |

### 横向步骤条

- 4 个步骤节点，节点间有连接线
- 状态：待完成（灰色）/ 当前激活（蓝色高亮 + 光晕）/ 已完成（蓝色实心）
- 点击任意节点可切换查看对应步骤内容
- 步骤内容区底部有「上一步 / 下一步」导航按钮

### 步骤内容区

每个步骤展示：
- 步骤标题 + 描述文字
- 处理结果图片（横向排列，自适应高度）
- 关键数据（公式、LAB 值、ΔE 等）

### 结果卡片（4 张，底部固定）

| 卡片 | 内容 |
|------|------|
| 左侧平均 LAB | L / a / b 竖向排列，作为基准参考色 |
| 右侧平均 LAB | L / a / b 竖向排列，反应区色值 |
| 整体 ΔE | 左右均值直接计算的色差值 |
| 超阈值比例 | 变色网格数 / 总网格数，百分比显示 |

---

## 步骤内容详情

### Step 1：ROI 提取
- 展示图片：二值化图、轮廓检测图、ROI 区域图
- 数据：ROI 尺寸

### Step 2：黑洞定位与裁剪
- 展示图片：黑洞定位标注图（红线为裁剪线，青色为黑洞轮廓）、裁剪结果图
- 数据：黑洞右边界 x 坐标、裁剪后尺寸

### Step 3：左右分割与色差计算
- 展示图片：左右分割示意图（绿色中线）
- 数据：CIE76 公式、左侧 LAB、右侧 LAB、整体 ΔE（大字突出显示）

### Step 4：网格色差与变色统计
- 展示图片：网格划分图、ΔE 热力图、变色标注图（红色高亮超阈值网格）
- 数据：阈值、超阈值网格数 / 总网格数、变色比例（大字突出显示）

## 视觉风格

| 元素 | 规格 |
|------|------|
| 主色 | 蓝色渐变 `#2563eb → #4f46e5` |
| Header | 深蓝渐变背景，高度 60px |
| 卡片 | 白色背景，圆角 12px，轻阴影 |
| 字体 | 系统默认 sans-serif；数值使用等宽字体 |
| 动画 | 步骤切换 fadeIn；卡片出现 fadeUp；加载进度条滑动 |

---

## 技术实现

| 层 | 技术 |
|----|------|
| 前端 | 原生 HTML5 + CSS3 + Vanilla JS，无框架依赖 |
| 后端 | Python Flask，端口 5002 |
| 通信 | REST API，JSON + Base64 图片传输 |
| 跨域 | flask-cors 处理 CORS |
